# Diretório `assets/gfx/icons`

Este diretório armazena todos os **ícones da interface**.

Ícones são pequenos elementos gráficos usados para representar ações, status ou categorias de forma visual e intuitiva.

Exemplos:
- Ícone de "Confirmar" (✓)
- Ícone de "Cancelar" (X)
- Ícone para a tela de "Configurações" (engrenagem)
- Ícones de navegação (setas)

Manter os ícones centralizados aqui facilita a reutilização e a aplicação de temas visuais.
