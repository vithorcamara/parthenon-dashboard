# Diretório `assets/gfx/logos`

Este diretório armazena arquivos de imagem para **logos diversos**.

Isso pode incluir:
- Logo do próprio frontend.
- Logos de sistemas (ex: PlayStation 2, PlayStation).
- Logos de publicadoras ou desenvolvedoras de jogos.
- Imagens de marca para aplicações.

Centralizar os logos aqui facilita o acesso e a manutenção da identidade visual do projeto.
