# Diretório `assets/gfx/backgrounds`

Este diretório contém as **imagens de fundo** para as diferentes telas (cenas/hubs) da aplicação.

Cada cena principal pode ter sua própria imagem de fundo para criar uma identidade visual única e contextual. Por exemplo, a `GameListScene` pode ter um fundo diferente da `SettingsScene`.

Os temas (`/assets/themes/*.json`) podem referenciar os arquivos neste diretório para permitir a customização dos fundos sem alterar o código.
