# Diretório `assets`

Este diretório contém todos os **dados estáticos e de configuração visual** do projeto. Nenhum arquivo localizado aqui deve conter lógica de programação.

A estrutura de subdiretórios organiza os diferentes tipos de assets:

- **gfx/**: Para imagens e elementos gráficos.
- **audio/**: Para arquivos de som e música.
- **fonts/**: Para fontes de texto.
- **themes/**: Para arquivos JSON que definem a aparência da interface.

Esta organização, conforme definida no SAD, permite que a aparência e os dados visuais do frontend sejam modificados sem a necessidade de alterar o código-fonte.
