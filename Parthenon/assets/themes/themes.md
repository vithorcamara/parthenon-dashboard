# Diretório `assets/themes`

Este diretório contém os arquivos de **tema da interface**, geralmente em formato `.json`.

Um tema é um arquivo de configuração que define a aparência visual da aplicação, permitindo que o frontend seja "reskinnable" (tenha sua aparência trocada) sem alterar o código-fonte.

Um arquivo de tema (`default.json`, por exemplo) pode definir:
- Paleta de cores (primária, destaque, fundo, texto).
- Nomes das fontes a serem usadas para títulos e corpo.
- Caminhos para imagens de fundo.
- Valores de espaçamento e layout.

Essa abordagem desacopla o design da lógica, alinhando-se perfeitamente com os objetivos da arquitetura.
