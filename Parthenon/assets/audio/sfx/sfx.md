# Diretório `assets/audio/sfx`

Este diretório contém os **efeitos sonoros (Sound Effects - SFX)** da interface do usuário.

Esses são sons curtos que fornecem feedback auditivo para as ações do usuário. Exemplos incluem:

- Som de navegação ao mover o foco entre itens.
- Som de confirmação ao pressionar o botão "Confirmar".
- Som de erro ao ocorrer um problema.
- Som de "voltar" ao cancelar uma ação.

Manter esses sons organizados aqui é essencial para uma experiência de usuário coesa e responsiva.
