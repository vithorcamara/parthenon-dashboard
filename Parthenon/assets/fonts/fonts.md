# Diretório `assets/fonts`

Este diretório armazena os arquivos de **fonte de texto** (ex: `.ttf`, `.otf`) utilizados no projeto.

A legibilidade é um pilar da UX vision, especialmente em telas de baixa resolução como as de CRTs. As fontes escolhidas devem ser claras, simples e funcionar bem em diferentes tamanhos.

Os arquivos de tema (`/assets/themes/*.json`) podem especificar qual fonte usar para cada tipo de texto (títulos, corpo, etc.), permitindo flexibilidade no design.
